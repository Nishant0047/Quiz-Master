# Quiz Master

A modern and interactive **Quiz Master** web application built using **HTML5, CSS3, and Vanilla JavaScript**. The application allows users to test their knowledge through multiple-choice quizzes, track their scores, review answers, and monitor their performance across different quiz categories.
This project focuses on JavaScript logic, state management, timers, JSON data handling, Local Storage, and responsive user interface development.

1. Project Overview

Quiz Master is an online quiz application designed to provide an engaging learning experience. Users can choose quiz categories, answer timed questions, review their results, and track their previous quiz performances.
The application dynamically loads questions from a JSON file and stores quiz history using the browser's **Local Storage**.
